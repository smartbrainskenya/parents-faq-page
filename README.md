# Parent Dashboard Help & Setup Guide

A comprehensive FAQ and setup guide for parents registering on the Smart Brains Kenya parent dashboard.

## Overview

This is a single-page HTML guide that helps parents through the complete setup process for accessing the Smart Brains Kenya parent dashboard. It covers:

- Finding their child's student username
- Creating a parent account
- Getting account approval
- Resolving SMS delivery issues
- Submitting home projects
- Frequently asked questions

## Features

- **Sticky Navigation Tabs** - Easy navigation between sections with active state tracking
- **Progressive Steps** - Clear step-by-step guides with visual indicators
- **Alert Boxes** - Contextual warnings, tips, and information boxes
- **Accordion FAQs** - Expandable FAQ section for quick answers
- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Modern UI** - Purple and yellow color scheme with smooth animations

## Sections

1. **Student Username** - How to find and verify your child's username
2. **Registration** - Step-by-step account creation guide
3. **Approval** - What to expect after registration
4. **SMS Issues** - Fixing blocked promotional messages
5. **Projects** - How to submit and manage home projects
6. **FAQs** - Common questions and answers

## Installation & Setup

No installation required. Simply open `parents-registration-FAQ-guide.html` in a web browser.

### Local Testing

1. Clone or download the repository
2. Open `parents-registration-FAQ-guide.html` in your browser
3. Navigate using the sticky tabs or scroll through sections

### Deploying

The file can be deployed to any web server. Ensure `hero-logo.png` is in the same directory as the HTML file.

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, flexbox, and gradients
- **Vanilla JavaScript** - Interactive FAQ accordion and smooth navigation
- **Google Fonts** - Plus Jakarta Sans and DM Sans typefaces

## Customization

### Colors

Edit the CSS variables in the `<style>` section:

```css
:root {
  --purple: #5B21B6;
  --yellow: #F59E0B;
  --green: #059669;
  /* ... other colors ... */
}
```

### Content

All text content is in the HTML. Edit directly in the `<section>` elements to update information.

### Logo

Replace `hero-logo.png` with your own logo image (recommended 32x32px).
